How I build blog site

How I built this blog site?

Basically, It’s a Github Page. I use Github Actions to automate the build process.

I made three repos:

  • blog: (private). It hosts only markdown files and images, which are the core contents of a blog site.

  • blog-site: (private). It’s the blog site source code repo, which would pull blog files to build a site.

  • XfLoops.github.io: (public). It’s a host repo for the output files built from blog-site.
